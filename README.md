# Que Pharma 💊

A web application that helps clinics notify patients when their medicine is ready — reducing waiting time and improving communication.

## 🌐 Live Demo

👉 [Open Que Pharma](https://que-pharma-production.up.railway.app/)

## 🚀 Problem

In many clinics, patients wait without clear updates on when their medicine is ready. This leads to:

* Long and uncertain waiting times
* Poor communication between staff and patients
* Inefficient patient flow

## 💡 Solution

Que Pharma allows clinics to:

* Update the status of prescriptions in real-time
* Notify patients directly through their mobile browser
* Send notifications via WhatsApp

Patients can simply:

* Access the web app on their phone
* Check their medicine status anytime
* Receive updates without needing to ask staff

## 🛠️ Tech Stack

* Python
* Django
* PostgreSQL (Railway)
* Gunicorn
* Whitenoise
